\chapter{\emph{Add-ons}}\label{cha:add-ons}

La \yatcl{} fournit des \emph{add-ons} destinés à faciliter son usage avec
différents éditeurs de texte.

\section{TeXstudio}
\label{sec:texstudio}

Le fichier de complétion \file{yathesis.cwl}, destiné l'éditeur
\href{http://texstudio.sourceforge.net/}{TeXstudio} se trouve dans le
répertoire \directory{.../tex/latex/yathesis/addons/completion/}. En attendant
que ce fichier soit officiellement avec cet éditeur, il suffit de le placer
dans le dossier :
\begin{description}
\item[sous Linux, Unix, Mac OS X :] \directory{~/.config/texstudio} ;
\item[sous Windows :] \directory{C:\Documents and Settings/User/AppData/Roaming/texstudio}.
\end{description}
